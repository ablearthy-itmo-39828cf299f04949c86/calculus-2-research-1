\section{Задача № 1}
Дана кусочно-заданная функция:
\startformula
g(x) =
  \startmathcases
    \NC \sqrt{x - 3},    \NC x \ge 3 \NR
    \NC \frac{1}{x - 3}, \NC 0 \le x < 3 \NR
    \NC 3 x^2 + 1,       \NC x < 0 \NR
  \stopmathcases
\stopformula

\startitemize[n]
\item Найдите такую непрерывную функцию \m{f(x)}, что \m{f'(x) = g(x)} или докажите, что она
    не может быть непрерывна.
\item Постройте графики функций \m{f(x)} и \m{g(x)} в одном масштабе и расположите их один под другим.
\item Проанализируйте графики, сделайте комментарии о виде графика функции \m{f(x)} в зависимости от вида графика функции \m{g(x)}.
\stopitemize
